Manual Testing Project: Bangladesh Railway E-Ticketing System
This repository contains a comprehensive manual testing suite for the Bangladesh Railway (E-Ticketing) Registration Page.
The project focuses on ensuring the reliability, security, and usability of the user registration flow, specifically focusing on 
NID and Birth Certificate verification processes.

📌 Project Overview
Project Name: E-Ticket-Bangladesh Railway
Module Under Test: Registration (NID and Final Sections)
Tester: Al Shahriar Refath
Testing Type: Manual Functional Testing, Negative Testing, Usability Testing
Environment: Google Chrome (Production Environment)

🎯 Objectives
The primary goal was to validate that the registration system:
Correctly verifies National ID (NID) credentials.
Forces Birth Certificate authentication if NID verification is bypassed.
Implements strict data validation for fields like Mobile Number, Email, and Date of Birth.
Provides a seamless and error-free user experience for genuine passengers.

📂 Repository Structure
Test Scenarios: High-level testing goals covering core functionalities.
Test Case Document: Detailed execution steps, expected results, and actual outcomes for 31 test cases.
Bug Report: Detailed documentation of identified defects with reproduction steps and severity levels.
Test Metrics & Summary: Statistical breakdown of the testing cycle and overall quality assessment.
